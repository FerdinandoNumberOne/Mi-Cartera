Notas importantes:
1. Para cripto: usa tipo Cripto. BTC puede ser BTC-USD en Yahoo y BITSTAMP:BTCUSD en TradingView.
2. Para fondos y monetarios: lo más fiable es Fuente Manual con valor liquidativo/precio actual manual.
3. Yahoo puede fallar por CORS/cookies porque no es API oficial; Finnhub necesita API Key.
4. El cálculo fiscal es orientativo y no sustituye Renta Web.
