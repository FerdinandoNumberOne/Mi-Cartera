Notas importantes

1. El cálculo fiscal es orientativo.

2. Las ganancias positivas tributan de forma progresiva como ahorro.

3. El FX EUR/USD se aplica a posiciones en USD.

4. Los datos se guardan en este navegador mediante localStorage.
